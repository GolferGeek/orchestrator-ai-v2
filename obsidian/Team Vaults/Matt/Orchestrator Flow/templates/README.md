# Templates

## Overview

Templates provide reusable structures for common documents and configurations.

## Template Types

### Agent Templates
- Agent definition templates (YAML/JSON)
- Context/knowledge base templates
- Prompt templates
- Testing templates

### Workflow Templates
- Single-agent workflow templates
- Multi-agent orchestration templates
- Approval workflow templates
- Error handling templates

### Project Templates
- Effort templates
- Project templates
- Goal templates
- Task templates

### Documentation Templates
- Flow documentation templates
- Role guide templates
- Topic documentation templates
- Checklist templates

## Usage

Templates should be:
- **Copyable** - Easy to duplicate and customize
- **Comprehensive** - Include all necessary fields
- **Documented** - Explain each section
- **Examples** - Include example content
- **Versioned** - Track template changes
