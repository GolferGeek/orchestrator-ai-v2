---
description: Stage all changes, commit with a generated message, and optionally push if -push is specified
---

1. Run `git status` to see what has changed.
2. Run `git add .` to stage all changes.
3. Generate a concise and descriptive commit message based on the changes.
4. Run `git commit -m "your message"`.
5. Check if the original request included `-push` or an instruction to push.
6. If yes, run `git push`.
