## INSTRUCTIONS FOR INTAKE SPECIALISTS

### Purpose
This checklist ensures comprehensive and consistent intake for all personal injury matters. Complete all applicable sections during initial client contact. Missing information should be flagged for follow-up.

### Completion Guidelines
1. Complete all sections marked with an asterisk (*) - these are mandatory
2. Skip sections not applicable to the case type
3. Use additional pages if needed for detailed descriptions
4. Attach all available documentation
5. Review with supervising attorney before finalizing

### Intake Process Overview
1. Initial phone screening (10-15 minutes)
2. Detailed intake interview (30-45 minutes)
3. Conflicts check
4. Attorney review and case evaluation
5. Engagement decision
6. Client onboarding
