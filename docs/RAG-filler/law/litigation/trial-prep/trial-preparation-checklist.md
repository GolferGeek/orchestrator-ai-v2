## Article I: Pre-Trial Timeline Overview

### Section 1.1: Master Timeline

#### 1.1.1 Critical Deadlines Matrix

| Days Before Trial | Action Item | Rule |
|-------------------|-------------|------|
| 120 | Discovery cutoff (typical) | Scheduling Order |
| 90 | Expert depositions complete | Scheduling Order |
| 60 | Dispositive motions deadline | Scheduling Order |
| 45 | Motions in limine deadline | Local Rule |
| 30 | Pretrial disclosures (FRCP 26(a)(3)) | Rule 26 |
| 28 | Joint pretrial statement due | Local Rule |
| 21 | Objections to pretrial disclosures | Rule 26 |
| 14 | Final pretrial conference | Rule 16 |
| 7 | Trial briefs due | Local Rule |
| 5 | Exhibit/witness lists finalized | Local Rule |
| 3 | Jury instructions submitted | Local Rule |
| 1 | Final preparation | - |

#### 1.1.2 Countdown Checklist by Phase

**Phase 1: 120-90 Days Out**
- [ ] Complete all outstanding discovery
- [ ] Complete expert depositions
- [ ] Evaluate case for settlement
- [ ] Identify trial themes
- [ ] Begin exhibit organization

**Phase 2: 90-60 Days Out**
- [ ] File dispositive motions if warranted
- [ ] Respond to dispositive motions
- [ ] Draft motions in limine
- [ ] Organize deposition transcripts
- [ ] Identify potential trial witnesses

**Phase 3: 60-30 Days Out**
- [ ] Finalize witness list
- [ ] Prepare pretrial disclosures
- [ ] Draft joint pretrial statement
- [ ] Begin witness preparation
- [ ] Organize trial exhibits

**Phase 4: 30-14 Days Out**
- [ ] Serve pretrial disclosures
- [ ] File motions in limine
- [ ] Draft trial brief
- [ ] Prepare jury instructions
- [ ] Continue witness preparation

**Phase 5: 14-7 Days Out**
- [ ] Attend final pretrial conference
- [ ] Finalize witness order
- [ ] Complete exhibit notebooks
- [ ] Draft opening statement
- [ ] Prepare direct examinations

**Phase 6: Final Week**
- [ ] Final witness preparation
- [ ] Prepare cross-examination outlines
- [ ] Review all exhibits
- [ ] Pack trial materials
- [ ] Conduct mock presentations

## Article II: Pretrial Disclosures and Statements

### Section 2.1: Rule 26(a)(3) Pretrial Disclosures

#### 2.1.1 Required Content

At least 30 days before trial, disclose:

**Witnesses:**
- Name, address, telephone (if not previously provided)
- Designation: Will call / May call
- Summary of expected testimony

**Depositions:**
- Names of witnesses whose deposition testimony will be offered
- Transcript page/line designations (if available)

**Exhibits:**
- Identification of each exhibit
- Designation: Will offer / May offer
- Basis for admissibility (if challenged)

#### 2.1.2 Pretrial Disclosure Format

```
PLAINTIFF'S PRETRIAL DISCLOSURES UNDER RULE 26(a)(3)

I. WITNESSES

A. Witnesses Plaintiff Will Call:

1. [WITNESS NAME]
   Address: [ADDRESS]
   Expected to testify regarding: [SUMMARY]

2. [WITNESS NAME]
   Address: [ADDRESS]
   Expected to testify regarding: [SUMMARY]

B. Witnesses Plaintiff May Call:

1. [WITNESS NAME]
   Address: [ADDRESS]
   Expected to testify regarding: [SUMMARY]

II. DEPOSITION TESTIMONY

Plaintiff intends to offer testimony from the following depositions:

1. Deposition of [NAME], taken [DATE]
   Page __, lines __-__ through Page __, line __

III. EXHIBITS

A. Exhibits Plaintiff Will Offer:

[Ex. No.] | [Description] | [Bates Range]
---------|---------------|-------------
1        | Contract dated [DATE] | PLAINTIFF 0001-0010
2        | Email from [X] to [Y] re: [SUBJECT] | PLAINTIFF 0011

B. Exhibits Plaintiff May Offer:

[List as above]
```

## Article III: Witness Preparation

### Section 3.1: Witness Identification and Scheduling

#### 3.1.1 Witness Categories

| Category | Considerations |
|----------|----------------|
| Party Witnesses | Most control; prepare extensively |
| Friendly Fact Witnesses | Coordinate schedule; prepare testimony |
| Hostile Fact Witnesses | May need subpoena; prepare cross |
| Expert Witnesses | Coordinate schedule; review opinions |
| Rebuttal Witnesses | May be conditional; prepare flexibly |

#### 3.1.2 Witness Availability Matrix

Create for each witness:
- Contact information
- Available dates during trial
- Notice requirements
- Transportation needs
- Special accommodations
- Backup contact
- Subpoena status

### Section 3.2: Witness Preparation Sessions

#### 3.2.1 Preparation Timeline

| Timing | Activity | Duration |
|--------|----------|----------|
| 4-6 weeks before | Initial meeting; discuss testimony | 1-2 hours |
| 2-3 weeks before | Document review; substantive prep | 2-4 hours |
| 1 week before | Mock examination | 2-4 hours |
| Day before/morning of | Final review; logistics | 30-60 min |

#### 3.2.2 Preparation Session Agenda

**Session 1: Initial Meeting**
- [ ] Explain trial process
- [ ] Discuss role as witness
- [ ] Review basic facts
- [ ] Identify key topics
- [ ] Discuss schedule and logistics
- [ ] Address concerns

**Session 2: Substantive Preparation**
- [ ] Review all relevant documents
- [ ] Discuss anticipated questions
- [ ] Practice key testimony points
- [ ] Review deposition testimony (if any)
- [ ] Identify difficult areas
- [ ] Discuss cross-examination topics

**Session 3: Mock Examination**
- [ ] Conduct direct examination
- [ ] Conduct mock cross-examination
- [ ] Provide feedback
- [ ] Practice difficult questions
- [ ] Review demeanor and presentation
- [ ] Final preparation instructions

#### 3.2.3 Key Witness Instructions

Advise witnesses to:

1. **Tell the Truth**: Always; credibility is paramount
2. **Listen to the Question**: Answer only what is asked
3. **Think Before Answering**: Brief pause is appropriate
4. **Answer Simply**: Don't volunteer extra information
5. **Admit When Uncertain**: "I don't recall" is acceptable
6. **Review Documents Carefully**: Before answering about them
7. **Stay Calm**: Don't argue with cross-examiner
8. **Look at the Jury**: When answering (jury trials)
9. **Ask for Clarification**: If question is confusing
10. **Wait for Objections**: Allow lawyer to object if needed

## Article IV: Exhibit Management

### Section 4.1: Exhibit Organization System

#### 4.1.1 Numbering Convention

| Party | Prefix | Example |
|-------|--------|---------|
| Plaintiff | P-[XXX] or PX-[XXX] | P-001, PX-001 |
| Defendant | D-[XXX] or DX-[XXX] | D-001, DX-001 |
| Joint | JX-[XXX] | JX-001 |
| Demonstrative | [Party]-DEM-[XXX] | P-DEM-001 |

#### 4.1.2 Exhibit Database Fields

Create database tracking:
- Exhibit number
- Description
- Bates range (if applicable)
- Date of document
- Author/recipient
- Witness(es) to authenticate
- Designation (Will/May offer)
- Stipulated (Y/N)
- Objections received
- Status (Admitted/Pending)

### Section 4.2: Exhibit Preparation

#### 4.2.1 Document Exhibits

For each document exhibit:
- [ ] Ensure clean, readable copy
- [ ] Apply exhibit sticker/label
- [ ] Create multiple copies (court, witness, counsel, jury)
- [ ] Prepare enlarged copy if needed
- [ ] Identify authenticating witness
- [ ] Prepare foundation questions

#### 4.2.2 Demonstrative Exhibits

Types of demonstratives:
- Timelines
- Charts and graphs
- Diagrams
- Photographs
- Maps
- Animations (if approved)
- Models

**Preparation Checklist:**
- [ ] Accurate and fair representation
- [ ] Not argumentative
- [ ] Clear and visible
- [ ] Disclosed to opposing counsel
- [ ] Approved by court (if required)
- [ ] Technology tested

### Section 4.3: Exhibit Notebooks

#### 4.3.1 Trial Exhibit Binder Structure

**Binder 1: Master Set**
- Numerical index
- All exhibits in order
- Tabs for each exhibit

**Binder 2: Witness Copies**
- Organized by witness
- Only exhibits for that witness
- Index for witness

**Binder 3: Court/Jury Set**
- Clean copies for presentation
- Properly marked
- Ready for admission

#### 4.3.2 Electronic Exhibit Organization

If using trial presentation software:
- [ ] Load all exhibits into system
- [ ] Test functionality
- [ ] Create callouts/highlights as needed
- [ ] Organize by witness/topic
- [ ] Train team on system
- [ ] Have backup plan

## Article V: Motions in Limine

### Section 5.1: Common Motion Topics

#### 5.1.1 Plaintiff Motions

| Topic | Purpose |
|-------|---------|
| Exclude defendant's expert | Challenge qualifications/methodology |
| Exclude character evidence | Prevent improper character attacks |
| Exclude settlement discussions | FRE 408 |
| Exclude remedial measures | FRE 407 |
| Preclude mention of insurance | FRE 411 |
| Exclude prejudicial evidence | FRE 403 balancing |

#### 5.1.2 Defendant Motions

| Topic | Purpose |
|-------|---------|
| Exclude plaintiff's expert | Challenge qualifications/methodology |
| Exclude "golden rule" arguments | Improper jury appeal |
| Limit damages evidence | Challenge calculation/foundation |
| Exclude hearsay | Improper out-of-court statements |
| Exclude prior acts | FRE 404 |
| Preclude cumulative evidence | Efficiency |

### Section 5.2: Motion Preparation

#### 5.2.1 Motion in Limine Checklist

- [ ] Identify evidentiary issue
- [ ] Research applicable rules
- [ ] Find supporting case law
- [ ] Draft motion with specificity
- [ ] Attach supporting evidence
- [ ] Prepare proposed order
- [ ] File by deadline
- [ ] Prepare for hearing

#### 5.2.2 Motion Format

```
MOTION IN LIMINE TO EXCLUDE [DESCRIPTION]

[Party] respectfully moves the Court for an order excluding
[specific evidence or testimony] from trial on the grounds
that [legal basis].

I. FACTUAL BACKGROUND
[Relevant facts]

II. LEGAL ARGUMENT
[Applicable rules and case law]

III. CONCLUSION
[Relief requested]
```

### Section 5.3: Responding to Motions

#### 5.3.1 Response Strategy

For each motion in limine:
- [ ] Analyze legal merit
- [ ] Research counter-authority
- [ ] Determine importance to case
- [ ] Draft response
- [ ] Prepare for hearing
- [ ] Have contingency if motion granted

## Article VI: Trial Brief and Jury Instructions

### Section 6.1: Trial Brief

#### 6.1.1 Standard Components

**Introduction:**
- Brief case summary
- Trial themes
- Key issues

**Factual Background:**
- Undisputed facts
- Contested facts (with support)
- Timeline if helpful

**Legal Arguments:**
- Elements of claims/defenses
- Application of law to facts
- Key case authority

**Evidentiary Issues:**
- Anticipated disputes
- Positions on admissibility
- Supporting authority

**Conclusion:**
- Summary of position
- Relief requested

#### 6.1.2 Trial Brief Checklist

- [ ] Clear, persuasive writing
- [ ] Accurate citations
- [ ] Record references
- [ ] Consistent with pretrial filings
- [ ] Addresses key disputed issues
- [ ] Appropriate length
- [ ] Filed by deadline

#### 6.2.2 Instruction Preparation Checklist

## Article VII: Trial Materials Preparation

### Section 7.1: Opening Statement

#### 7.1.1 Opening Statement Structure

**Introduction (Hook):**
- Capture attention
- Theme introduction
- Promise of what evidence will show

**Factual Narrative:**
- Tell the story
- Introduce key witnesses
- Reference key evidence
- Build toward conclusion

**Legal Framework:**
- Elements to prove
- How evidence meets elements
- Damages overview

**Conclusion:**
- Reinforce theme
- Request verdict

#### 7.1.2 Opening Statement Preparation

- [ ] Draft written outline
- [ ] Identify key exhibits to reference
- [ ] Time the presentation
- [ ] Practice delivery
- [ ] Prepare visual aids (if using)
- [ ] Anticipate objections
- [ ] Have backup if technology fails

### Section 7.2: Direct Examination Outlines

#### 7.2.1 Outline Format

For each witness, prepare:

**Witness: [NAME]**

| Topic | Questions | Exhibits |
|-------|-----------|----------|
| Background | Q1-Q5 | None |
| Key Event | Q6-Q15 | Ex. 1, 2 |
| Damages | Q16-Q20 | Ex. 3 |

**Foundation Questions:**
- Authentication of exhibits
- Personal knowledge
- Business records (if applicable)

**Substantive Questions:**
- Chronological or topical
- Open-ended when possible
- Leading on preliminary matters only

#### 7.2.2 Direct Examination Tips

**DO:**
- Use simple, clear language
- Let the witness tell the story
- Pause for emphasis on key points
- Reference exhibits appropriately
- Listen to answers

**DON'T:**
- Lead on disputed facts
- Ask compound questions
- Argue with the witness
- Rush through testimony
- Read verbatim from script

### Section 7.3: Cross-Examination Preparation

#### 7.3.1 Cross-Examination Principles

- Have clear objectives (limited)
- Use leading questions
- Control the witness
- Know when to stop
- Never ask a question you don't know the answer to
- Have impeachment material ready

#### 7.3.2 Cross-Examination Outline

**Witness: [NAME]**

**Primary Goals:**
1. [Specific point to establish]
2. [Specific admission to obtain]
3. [Credibility point]

**Preparation:**
- Prior testimony excerpts (page/line)
- Documents for impeachment
- Prior inconsistent statements
- Bias/motive evidence

### Section 7.4: Closing Argument

#### 7.4.1 Closing Argument Structure

**Introduction:**
- Remind of burden/standard
- Restate theme
- Preview argument

**Evidence Review:**
- Walk through key evidence
- Connect to elements
- Highlight favorable testimony
- Address weaknesses

**Responding to Opponent:**
- Address key opposing arguments
- Distinguish unfavorable evidence
- Explain inconsistencies

**Jury Instructions:**
- Reference applicable instructions
- Apply law to facts
- Explain verdict form

**Conclusion:**
- Call to action
- Request specific verdict

#### 7.4.2 Closing Argument Preparation

- [ ] Outline main points
- [ ] Reference specific testimony
- [ ] Reference key exhibits
- [ ] Tie to jury instructions
- [ ] Practice timing
- [ ] Prepare rebuttal points
